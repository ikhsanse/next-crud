# React Post Management
This is a simple React application for managing posts. It allows users to view, create, edit, and delete posts.

# Technologies 
- Next JS
- React Hook Form
- React Query
- Axios
- Chakra UI
# Features

- View a list of posts
- Add a new post
- View post details
- Edit a post
- Delete a post
